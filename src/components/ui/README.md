# UI Components

This folder contains shadcn/ui components.

## Adding Components

To add shadcn/ui components, use:

```bash
npx shadcn@latest add <component-name>
```

For example:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
```

## Available Components

Visit [ui.shadcn.com](https://ui.shadcn.com/docs/components) to see all available components.

## Current Components

Currently, this project uses custom components in the `shared/` folder. You can add shadcn/ui components here as needed for future features.
